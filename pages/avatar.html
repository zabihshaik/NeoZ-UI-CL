Avatar

Avatars are used for profile pictures, usually found in circular shapes. Avatars are commonly used with gravatar, profile pages, person list.

Avatars of Different Sizes

You can use images in avatars. You need to include class avatar and for required size, add class name according to the size. For eg. avatar-lg for large sized avatar, avatar-md for medium sized avatar, avatar-sm for small sized avatar, and avatar-xs for extra small sized avatar as shown below.
